# [Ngữ pháp]
* HTD: S + V1
* QKD: S + V2
* TLD: S + will + V0

* Tiếp diễn   : tobe + V_ing
* Hoàn thành  : have + V3
* Bị động     : tobe + V3

# [Ghi chú]
* V1 chia theo hiện tại
* V2 chia theo quá khứ
* V3 quá khứ phân từ
* Một số *...* luôn *đứng sau V_tobe* và *đứng trước V_thường*

* Ví dụ:
V0    => V1    =>    V2   =>    V3
be       am/is/are   was/were   been
